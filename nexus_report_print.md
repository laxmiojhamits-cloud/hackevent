## 🚀 The Vision
**NexusHack 2026** is the flagship hackathon platform for **MITS Gwalior**. Designed with a premium, editorial "Sky Blue" aesthetic, it prioritizes a smooth user journey from information discovery to final registration.

### ✨ Key Professional Features:
- **Sky Blue Design Language**: A minimalist, high-contrast theme.
- **Smart Registration Engine**: Real-time validation (React-Hook-Form + Zod).
- **MITS Gwalior Hub**: Full localization for the Gwalior venue.
- **Performance Optimized**: Built on Next.js 14 for speed and SEO.

---

## 🛠️ Technical Stack
- **Framework**: Next.js 14 (App Router)
- **Database**: Prisma ORM with SQLite (Reliable local storage)
- **Styling**: Tailwind CSS + Shadcn UI
- **Authentication**: NextAuth.js
- **Deployment**: Netlify
